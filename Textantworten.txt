Aufgabe B.4.1.5

(Ein Name muss mit einem Buchstaben oder einem Unterstrich beginnen und darf anschließend beliebig viele Buchstaben, 
Unterstriche und Ziffern enthalten. Das erste Zeichen darf keine Ziffer sein, damit der Compiler einen Namen leichter von einer Zahl 
unterscheiden kann. Die Buchstaben können klein oder groß sein.)

Welche der folgenden Variablennamen sind unzulässig, werden vom Compiler also nicht akzeptiert?


1. variablenname

Antwort: zulässig.


2. 2be_or_not_to_be

Antwort: nicht zulässig. Eine Variable darf nicht mit einer Zahl anfangen.


3. vorname.nachname

Antwort: nicht zulässig. Die Variable darf keinen Punkt enthalten, da dieser Punkt als Operator aufgefasst wird.


4. _system

Antwort: zulässig.


5. #hashtag

Antwort: nicht zulässig Eine Variable muss mit einem Buchstaben oder einem 
Unterstrich anfangen. Befehle mit # sind Präprozessor Anweisungen.


6. SiebenMeilenStiefel

Antwort: zulässig.


Aufgabe B.4.1.6

Bevor man eine Variable in einem C++-Programm definiert, muss man ihren Datentyp angeben. Wie
heißen die Schlüsselwörter zur Definition von Variablen der folgenden Typen:

1. Gleitpunktzahlen
Antwort: float oder double

2. Ganzen Zahlen
Antwort: int

3. Zeichen
Antwort: char

4. Strings
Antwort: ""

5. Booleschen Werten
Antwort: bool

6. Natürlichen Zahlen
Antwort: unsigned
